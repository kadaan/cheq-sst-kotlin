Models

class Models

The Models class provides a way to manage and access models that collect and expose data. Custom models can be added to the collection as long as they do not conflict with base models. The following base models are always included and cannot be overridden:

  • AppModel

  • DeviceModel

  • LibraryModel

class CustomModel : Model<CustomModel.Data>(Data::class, "custom_model") {
    override suspend fun get(modelContext: ModelContext): Data {
        return Data()
    }

    inner class Data : Model.Data() {
        @JsonProperty
        val foo: String = "bar"
    }
}

// Create the Models collection with the custom model
val models = Models(CustomModel())
println(models.all().size) // 4
println(models.get<CustomModel>()?.identifier?.key) // custom_model 
   //sampleEnd
}
