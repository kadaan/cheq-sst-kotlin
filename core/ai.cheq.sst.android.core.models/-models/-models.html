Models

constructor(vararg customModels: Model<*>)

Creates a new collection of models including the specified custom models.

Parameters

customModels

Custom models to be added to the collection.

Throws

InvalidModelException

If a base model with the same key already exists.

DuplicateModelException

If custom model with the same key or type has already been added.
